JL EXPRESS LOGÍSTICA — SITE

COMO ABRIR:
1. Abra a pasta jl-express-site no Visual Studio Code.
2. Clique duas vezes em index.html para abrir no navegador.
3. Para recarregar automaticamente, instale a extensão "Live Server" no VS Code.
4. Clique com o botão direito em index.html e escolha "Open with Live Server".

ARQUIVOS PRINCIPAIS:
- index.html: textos e estrutura do site.
- styles.css: cores, tamanhos e visual.
- script.js: calculadora, formulários, menu e WhatsApp.
- assets/: logotipo e imagens.

ATENÇÃO:
A calculadora mostra o volume e envia os dados para análise. Ela ainda não calcula o preço exato do frete, pois faltam as regras comerciais da transportadora.
